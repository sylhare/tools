# Tools

Webapp for simple tools.


## Installation

```bash
pnpm install
```

## Development

```bash
# Start development server (http://localhost:3000)
pnpm run dev

# Build for production
pnpm run build

# Preview production build
pnpm run preview
```

## Testing

```bash
# Run all tests
pnpm run lint && pnpm run type-check && pnpm run test && pnpm run test:e2e

# Linting
pnpm run lint
pnpm run lint:fix

# Type checking
pnpm run type-check

# Unit tests
pnpm run test
pnpm run test:watch
pnpm run test:coverage

# E2E tests
pnpm run test:e2e
pnpm run test:e2e:ui
```
